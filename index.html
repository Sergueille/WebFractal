Comment sont générées les fractales? Pour chaque point du plan C(x, y), on considère une suite de complexes ZC. L'ordinateur va calculer le nombre d'éléments spécifiés, et déterminera si la suite tend vers l'infini. Voici les formules utilisées:
Mandelbrot:

Z0 = z0

Zn+1 = Zn2 + C

Julia:

Z0 = C

Zn+1 = Zn2 + D

Burning ship:

Z0 = z0

Zn+1 = (|Re(Zn)| + i|Im(Zn)|)2 + C

Mandelbrot modifié:

Z0 = z0

Zn+1 = Zn2 + C3 + δ
